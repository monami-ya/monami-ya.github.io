PizzaFactory Camino - Cloud IDE Hosting

PizzaFactory Camino は，クラウド IDE のホスティングサービスです。

合同会社もなみ屋が管理しているサーバクラスタ内に，お客様専用の名前空間を設定し運用いたします。

機能概要

  • Eclipse/Che ベース
  • 遠隔ペアプログラミングを支援
  • Camino 独自のコンテナやプラグインを提供
  • セキュリティ
  • BASIC AUTH パスワード認証
  • 軽快な応答
  • サーバクラスタは Microsoft Azure の東日本リージョンに配置
  • 日本語によるサポート

Eclipse/Che ベース

基盤となるクラウド IDE として，Eclipse プロジェクトが開発を推進している Eclipse/Che を採用。 安定した開発保守体制の恩恵を受けられます。

遠隔ペアプログラミングを支援

Camino の基盤である Eclipse/Che は，複数開発者による同時編集をサポートしています。 複数開発者が同時に同じワークスペースを開くことができ，同時に編集できます。 編集は即時に保存されるため，競合は起きません。

Camino 独自のコンテナやプラグインを提供

10年以上 Eclipse 関連製品をリリースしてきた PizzaFactory チームの手によるプラグインが Che にインストールされた状態で提供されます。 お客様はインフラや開発ツール周りの環境整備に煩わされること無く，アプリケーション開発に集中して頂けます。 現在追加を予定している機能は，下表のとおりです。

セキュリティ

Camino ワークスペースとお手元のブラウザとの間は secure http (https) で通信されます。中間の proxy などでの傍受は困難です。

Camino ワークスペースは BASIC AUTH パスワード認証により護られ，第三者からの不正なアクセスを防ぎます。

軽快な応答

サーバクラスタは，Microsoft Azure の東日本リージョンに配置されています。

他社クラウド IDE の多くは日本国外にサーバがあり，通信の安定性や応答速度に不安がある場合がしばしばあります。 Camino は，日本のお客様を特に重視しています。

日本語によるサポート

サポートは日本人が行います。万が一の機材トラブル発生時でも，日本語でご連絡いただけます。

価格と購入方法

グレードは，下記の3つの要素でお選び頂けます。

  • Workspace 実行サーバを他のお客様と共有するか否か
  • お使いになるリソースの大きさ
  • アカウントが有効な日数

お申し込み頂いてから開通まで3営業日ほどお時間を頂く場合があります。日数は開通日の翌日0:00:00(JST)を1日目としてカウントいたします。

いずれのサイズも 30日，90日のいずれかをご選択いただけます。
