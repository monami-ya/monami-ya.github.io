FatFsPro

FatFsPro は，オープンソースの組み込み向けファイルシステム関連ソフトウェア群に対し，合同会社もなみ屋独自の検証，バグ修正，機能拡張を行ったミドルウェアです．

FatFsPro は，下記のソフトウェアを含みます．

  • FatFs
  • TOPPERS for FatFs

FatFs は，ELM氏が開発・公開しており，以下の様な特徴を持っています．

  • 市販製品を含む，多数の動作実績
  • 1チップマイコンでも動作する少リソース性
  • 1条項BSDライセンスと同等の自由度の高いライセンス

TOPPERS for FatFs は，FatFs を TOPPERS/JSP カーネルで動作させるためのパッケージです． TOPPERS for FatFs は，TOPPERSプロジェクトが配布しており，以下の様な特徴を持っています．

  • 市販製品を含む，多数の動作実績
  • 1チップマイコンでも動作する少リソース性
  • TOPPERSプロジェクトへの報告を行うことで，ソースコード開示が不要

FatFsPro は，上記のソフトウェアのライセンスを尊重し，新たに機能を付加する場合でも再配布を妨げることがないようにライセンスを設定しています．

FatFsPro は，その全てを git リポジトリとして公開しています．

FatFsPro サポートサービス

合同会社もなみ屋では，組込みシステムのプロフェッショナルとして，FatFsPro のエンジニアリングサポートを行っています． FatFsPro サポートサービスは，製品版の FatFsPro をご購入の方のみご利用できます．

無償のオープンソース製品に対して，有償のサポートを購入するメリット・デメリットにつきましては，別ページ もご参照ください．

FatFsPro サポートサービスのサービスレベルアグリーメント(SLA) もご参照ください．
