      31             :   == Object's Color
      32             : 
      33             :   Each object to be painted in three colors.
      34             : 
      35             :     * White - Unmarked.
      36             :     * Gray - Marked, But the child objects are unmarked.
      37             :     * Black - Marked, the child objects are also marked.
      38             : 
      39             :   == Two white part
      40             : 
      41             :   The white has a different part of A and B.
      42             :   In sweep phase, the sweep target white is either A or B.
      43             :   The sweep target white is switched just before sweep phase.
      44             :   e.g. A -> B -> A -> B ...
      45             : 
      46             :   All objects are painted white when allocated.
      47             :   This white is another the sweep target white.
      48             :   For example, if the sweep target white is A, it's B.
      49             :   So objects when allocated in sweep phase will be next sweep phase target.
      50             :   Therefore, these objects will not be released accidentally in sweep phase.
